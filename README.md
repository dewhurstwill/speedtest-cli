# Speedtest Service
---

## CLI
---

### Install the CLI
```bash
$ git clone git@github.com:dewhurstwill/speedtest-service.git
$ cd speedtest-service
$ npm run install-cli
```

### Uninstall the CLI
```bash
$ git clone git@github.com:dewhurstwill/speedtest-service.git
$ cd speedtest-service
$ npm run uninstall
```

### Using the CLI
```bash
$ speedtest-cli --help
Usage:
speedtest-cli run [--debug -d -D] [--help -h -H]
or
speedtest-cli help

$ speedtest-cli run
```
